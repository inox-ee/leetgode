```dot
digraph G {
  A [
    label="[ ]\n[1,2,3]"
  ]
  B [
    label="[1]\n[2, 3]"
  ]
  C [
    label="[2]\n[1, 3]"
  ]
  D [
    label="[3]\n[1, 2]"
  ]
  BB [
    label="[1, 2]\n[3]"
  ]
  BC [
    label="[1, 3]\n[2]"
  ]
  BBB [
    label="[1, 2, 3]\n[ ]"
  ]
  BCB [
    label="[1, 3, 2]\n[ ]"
  ]
  A -> {B, C, D}
  B -> {BB, BC}
  BB -> BBB
  BC -> BCB
  BBB -> B [ label="ここ" ]
}
```
