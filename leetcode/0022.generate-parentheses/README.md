## 解答例１

- 解法：backtracking

backtracking により、正誤判定をしながら全通り探索できる。
思いついてはいたが、再帰という発想がなかった…

### backtracking とは

制約充足問題を探索する戦略の一種。一種の深さ優先探索とも言える。
一般に **再帰呼び出し関数** として実装される。

代表例：0/1 ナップサック問題、N クイーン問題
